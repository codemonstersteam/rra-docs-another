# intent — rra-docs-another

Оценить качество документации **произвольного** git-репозитория для работы
четырёх JTBD-потребителей (инженер-создатель, инженер-потребитель, менеджер,
ИИ-агент) и выдать четыре независимых score плюс список конкретных пробелов.

## Контекст

Универсальная и более сложная проверка, чем гейт `rra-docs`. Не предполагает
рациональную дисциплину и её артефакты — работает на любом репо с гитхаба.
Шестислойная оценка L1–L6, ИИ только на верхних слоях (L5 и опциональный L6c).

## Не-цели

- Не проверяет соответствие дисциплине из скиллов — это узкий гейт `rra-docs`.
- Не линтер орфографии и не счётчик покрытия.
- Не усредняет JTBD в одну цифру.
- Не аудитит I/O-границу кода — это критерий №1 (`rra-audit-repo`).
